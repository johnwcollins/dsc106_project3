Interactive Blood-Pressure Visualization

Project Write-Up

Design Rationale

Our goal was to visualize how rats’ blood pressure changes over time under different salt intake conditions. We chose a line chart because it clearly shows time-series patterns. To keep the visualization interactive but focused, we added group and rat selectors. The trendline helps identify long-term increases or decreases. Tooltips and a vertical crosshair provide details on demand. We considered showing all rats at once, but it became visually cluttered, so we focused on one rat at a time with flexible filters.

Development Overview

We used D3.js to build the chart, loading a pre-processed JSON file of blood pressure readings. We implemented dropdown filters, a trendline toggle, tooltips for both data points and trendlines, a crosshair for time alignment, and axis labels for clarity.

  • Time Spent: ~20–25 person-hours total
  • Most Time-Consuming Part: Implementing smooth interaction for trendlines and tooltips without cluttering the UI
